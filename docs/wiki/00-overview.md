# NearPick — 프로젝트 개요

## 서비스 한 줄 설명

> 근처 가게의 인기 상품을 실시간으로 탐색하고, 찜·예약·선착순 구매까지 한 번에.

---

## 사용자 유형

| 유형 | 역할 |
|------|------|
| **소비자 (Consumer)** | 근처 인기 상품 탐색 → 찜 / 예약 / 선착순 구매 |
| **소상공인 (Merchant)** | 상품·할인권 등록 → 노출 / 판매 관리 |
| **관리자 (Admin)** | 플랫폼 전체 모니터링 · 사용자·상품 관리 |

---

## 핵심 기능

### 소비자

- **근처 상품 탐색**: 현재 위치 기반 반경 내 인기 상품 조회
- **찜 (Wishlist)**: 관심 상품 저장
- **예약 (Reservation)**: 방문 수령 일시 예약
- **선착순 구매 (Flash Purchase)**: 한정 수량 시간 내 구매

### 소상공인

- **상품 등록**: 일반 상품(GENERAL) / 선착순 상품(FLASH_SALE) 구분 등록
- **할인권 등록**: 기간·수량 한정 할인 설정
- **판매 현황 조회**: 예약·구매 현황 모니터링

### 관리자

- **사용자 관리**: 소비자·소상공인 계정 관리, 정지·강제 탈퇴
- **상품 검수**: 부적절 상품 강제 종료(FORCE_CLOSED)
- **플랫폼 모니터링**: 인기점수 배치, 거래 통계

---

## 비즈니스 규칙

1. **배송 없음**: 모든 거래는 소비자가 직접 가게를 방문하여 수령
2. **인기점수**: `PopularityScore` — 찜 수, 예약 수, 구매 수를 가중 합산, 배치(batch)로 갱신
3. **상품 상태 흐름**:
   ```
   ACTIVE → SOLD_OUT | CLOSED | FORCE_CLOSED
   SOLD_OUT → ACTIVE (재고 보충 시)
   ```
4. **선착순 구매**: `FlashPurchase` — 수량 소진 시 자동 SOLD_OUT 처리
5. **예약**: `Reservation` — 소비자가 방문 일시 선택, 소상공인이 확인
6. **위치 정보**: 위도/경도(DECIMAL 10,7)로 저장, 향후 PostGIS 마이그레이션 가능
