## CON — 소비자 화면

### CON-01: 홈 — 인기 상품 목록

```
┌─────────────────────────────────┐
│  📍 강남구 역삼동          [검색] │
│  ─────────────────────────────  │
│  근처 인기 상품                  │
│                                 │
│  ┌────────┐  ┌────────┐         │
│  │ 이미지 │  │ 이미지 │          │
│  │        │  │        │         │
│  │상품명  │  │상품명  │          │
│  │가격    │  │가격    │          │
│  │⭐ 인기  │  │⭐ 인기  │         │
│  │🏷 FLASH │  │        │         │
│  └────────┘  └────────┘         │
│                                 │
│  [더 보기]                       │
└─────────────────────────────────┘
```

**표시 데이터**:
- 현재 위치 (위도/경도)
- 상품 카드: `id`, `title`, `price`, `productType`, `popularityScore`, 대표 이미지
- 페이지네이션: `page`, `size`, `totalCount`

**필터**: 반경(km), 상품 유형(GENERAL/FLASH_SALE), 정렬(인기순/거리순)

**액션 → API**:
| 액션 | API |
|------|-----|
| 화면 진입 (위치 기반) | `GET /products/nearby?lat=&lng=&radius=&sort=` |
| 상품 카드 클릭 | → CON-02 |

---

### CON-02: 상품 상세

```
┌─────────────────────────────────┐
│  ← 뒤로                          │
│                                 │
│  [상품 이미지]                    │
│                                 │
│  상품명                ♡ 찜      │
│  ₩ 가격                          │
│  📍 가게명 · 주소 · 거리          │
│                                 │
│  ─ 상품 설명 ──────────────────  │
│  상세 설명 텍스트...              │
│                                 │
│  ─ 인기점수 ───────────────────  │
│  ♡ 찜 12   📅 예약 5   🛒 구매 3  │
│                                 │
│  [FLASH] 남은 수량: 8개           │
│  [예약하기]  [선착순 구매]         │
└─────────────────────────────────┘
```

**표시 데이터**:
- `id`, `title`, `description`, `price`, `productType`, `status`
- `shopLat`, `shopLng`, `shopAddress` (가게 위치)
- `popularityScore`: `wishlistCount`, `reservationCount`, `purchaseCount`
- (FLASH_SALE) `remainingQuantity`

**액션 → API**:
| 액션 | API |
|------|-----|
| 화면 진입 | `GET /products/{id}` |
| 찜 추가 | `POST /wishlists` `{ productId }` |
| 찜 제거 | `DELETE /wishlists/{productId}` |
| 예약하기 클릭 | → CON-03 |
| 선착순 구매 클릭 | → CON-04 |

### CON-03: 예약하기

```
┌─────────────────────────────────┐
│  ← 예약하기                      │
│                                 │
│  상품명                          │
│  ₩ 가격                          │
│                                 │
│  방문 날짜  [2026-03-01 ▼]       │
│  방문 시간  [14:00 ▼]            │
│                                 │
│  수량       [1 ▼]               │
│                                 │
│  요청 사항  [메모 입력...]        │
│                                 │
│  ─────────────────────────────  │
│  총 금액    ₩ 12,000             │
│                                 │
│  [예약 확정]                      │
└─────────────────────────────────┘
```

**표시 데이터**: 상품명, 가격 (CON-02에서 전달)

**액션 → API**:
| 액션 | API |
|------|-----|
| 예약 확정 | `POST /reservations` `{ productId, visitAt, quantity, memo }` |
| 성공 | → 예약 완료 토스트 + CON-05b 이동 |

## Phase 4 API 목록 도출

Phase 3 목업에서 도출된 필요 API 전체 목록:

### 인증 (Auth)
| Method | Path | 설명 |
|--------|------|------|
| POST | `/auth/login` | 로그인 |
| POST | `/auth/signup/consumer` | 소비자 회원가입 |
| POST | `/auth/signup/merchant` | 소상공인 회원가입 |

### 상품 (Product)
| Method | Path | 설명 |
|--------|------|------|
| GET | `/products/nearby` | 위치 기반 인기 상품 목록 |
| GET | `/products/{id}` | 상품 상세 |
| POST | `/products` | 상품 등록 (소상공인) |
| PATCH | `/products/{id}/close` | 상품 종료 (소상공인) |
| GET | `/products/me` | 내 상품 목록 (소상공인) |

### 찜 (Wishlist)
| Method | Path | 설명 |
|--------|------|------|
| POST | `/wishlists` | 찜 추가 |
| DELETE | `/wishlists/{productId}` | 찜 제거 |
| GET | `/wishlists/me` | 내 찜 목록 |

### 예약 (Reservation)
| Method | Path | 설명 |
|--------|------|------|
| POST | `/reservations` | 예약 생성 |
| GET | `/reservations/me` | 내 예약 목록 (소비자) |
| PATCH | `/reservations/{id}/cancel` | 예약 취소 |
| GET | `/reservations/merchant` | 예약 목록 (소상공인) |
| PATCH | `/reservations/{id}/confirm` | 예약 확정 (소상공인) |

### 선착순 구매 (FlashPurchase)
| Method | Path | 설명 |
|--------|------|------|
| POST | `/flash-purchases` | 선착순 구매 |
| GET | `/flash-purchases/me` | 내 구매 내역 |

### 소상공인 대시보드
| Method | Path | 설명 |
|--------|------|------|
| GET | `/merchants/me/dashboard` | 대시보드 요약 |

### 관리자 (Admin)
| Method | Path | 설명 |
|--------|------|------|
| GET | `/admin/users` | 전체 사용자 조회 |
| PATCH | `/admin/users/{id}/suspend` | 계정 정지 |
| DELETE | `/admin/users/{id}` | 강제 탈퇴 |
| GET | `/admin/products` | 전체 상품 조회 |
| PATCH | `/admin/products/{id}/force-close` | 상품 강제 종료 |
